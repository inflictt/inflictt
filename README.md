## 🚀 Selected Work - *Things I've shipped, not just started*

### [AI-FinOps: AWS Cost Intelligence Platform](https://github.com/inflictt/Ai-Finops) `internship project`
Built end to end during my Kansoft internship. Google Gemini converts raw AWS cloud-cost data into natural-language summaries and prioritized cost-optimization recommendations. Automated weekly reporting with node-cron and branded PDF generation via pdf-lib, replacing a fully manual process.
`React · Node.js · Express · Neon · Google Gemini`

### [Portfolio v2: inflict.in](https://github.com/inflictt/Portfolio-inflict.in) `live`
The site's entire background is a real-time WebGL cloud shader - every pixel computed each frame from layered noise, no animation libraries. Ink-blob intro, scroll-scrubbed falling project deck, a snaking SVG timeline, View Transitions theme sweep, live GitHub + LeetCode stats, GA4 event tracking and a 100 Lighthouse SEO score.
`React 19 · Vite · Tailwind v4 · WebGL · GLSL`

### [ATS-Ranker: AI Resume Screening](https://github.com/inflictt/ATS-Ranker)
ML-powered resume-ranking platform - REST APIs for PDF upload and job-description parsing, with NLP similarity scoring (TF-IDF vectorization, cosine matching) for real-time candidate ranking.
`Python · Django REST Framework · NLP · SQLite`

### [SneakPeak: Sneaker Marketplace](https://github.com/inflictt/SneakPeak-BML) `I&E Compendium`
Full-stack price-comparison marketplace covering 50+ verified vendors with secure auth, session persistence and cart CRUD. Selected as an official study resource for the university's 2025-26 Innovation & Entrepreneurship Compendium.
`Node.js · Express · MongoDB · EJS`

### [Typebeat](https://github.com/inflictt/Typebeat)
A minimal typing trainer in React - live speed and accuracy, keystroke by keystroke, zero clutter.
`React · JavaScript`
